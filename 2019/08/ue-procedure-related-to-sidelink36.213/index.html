UE procedure for transmitting PSCCH

For sidelink transmission mode 3, the UE shall determine the subframes and resource blocks for transmitting SCI format 1 as follows: + SCI format 1 is transmitted in two PRBs per slot in each subframe where the corresponding PSSCH is transmitted + If UE receives DCI format 5A in subframe $n$ with CRC scrambled by SL-V-RNTI, one transmission of PSCCH is in the PSCCH resources $L{Init}$ in the first subframe that is included in $(t{0}^{SL},t{1}^{SL},t{2}^{SL},…))$ and that starts not earlier than $T{DL}-\frac{N{TA}}{2}\cdot Ts+(4+m)\cdot 10^{-3}$. + If “Time gap between initial transmission and retransmission” in the configured sidelink grant is not equal to zero, another transmission of PSCCH is in the PSCCH resource $L{ReTX}$ in the subframe $t{q+SF{gap}}^{SL}$, where $SF_{gap}$ is the value indicated by “Time gap between initial transmission and retransmission” filed in the configured sidelink grant, subframe $tq^{SL}$ corresponds to the subframe $n+k{init}$. $L{ReTX}$ corresponds to the value $n{subCH}^{start}$ determined by the RIV set to the value indicated by “Frequency resource location of initial transmission and retransmission” field in the configured sidelink grant. + If the UE receives DCI format 5A in subframe $n$ with the CRS scrambled by SL-SPS-V-RNTI, and if the received DCI activates an SL SPS configuration, one transmission of PSCCH is in the resource $L{Init}$ is in the first subframe that is included in $(t{0}^{SL},t{1}^{SL},t{2}^{SL},…))$ and that starts not earlier than $T{DL}-\frac{N{TA}}{2}XT_s+(4+m)\cdot10^{-3}$.

SL index field in DCI format 5A Indicated value m
‘00’ 0
‘01’ 1
‘10’ 2
‘11’ 3
  • If “Time gap between initial transmission and retransmission” in the configured sidelink grant is not equal to zero, another transmission of PSCCH is in the PSCCH resource $L{ReTX}$ in the subframe $t{q+SF{gap}}^{SL}$, where $SF{gap}$ is the value indicated by “Time gap between initial transmission and retransmission” filed in the configured sidelink grant, subframe $tq^{SL}$ corresponds to the subframe $n+k{init}$. $L{ReTX}$ corresponds to the value $n{subCH}^{start}$ determined by the RIV set to the value indicated by “Frequency resource location of initial transmission and retransmission” field in the configured sidelink grant.
  • For other contents of SCI format 1
    • Set Modulation and coding scheme as indicated by high layers
    • “Priority” field is set according to the highest priority among those prioity(s) indicated by higher layers corresponding to the transport block
    • “Resource reservation” sets to zero
    • “Time gap between initial transmission and retransmission”,“Frequency resource location of initial transmission and retransmission”,“Retransmission index” are set according to the resources determined for PSSCH.
  • UE shall randomly select the cyclic shift $n_{cs,\lambda}$ among {0,3,6,9} in each PSCCH transmission

For sidelink transmission mode 4, - The UE shall determine the subframes and resource blocks for transmitting SCI format 1 as follows: - SCI format 1 is transmitted in two physical resource blocks per slot in each subframe where the corresponding PSSCH is transmitted. - If the configured sidelink grant from higher layer indicates the PSCCH resource in subframe $t_n^{SL}$, one transmission of PSCCH is in the indicated PSCCH resource m in subframe $tn^{SL}$ - If “Time gap between initial transmission and retransmission” in the configured sidelink grant is not equal to zero, another transmission of PSCCH is in the PSCCH resource $L{ReTX}$in subframe $t{n+SF{gap}^{SL}}$ where $SF{gap}$ is the value indicated by “Time gap between initial transmission and retransmission” field in the configured sidelink grant,$L{ReTX}$ corresponds to the value $n_{start}^{subCH}$ determined by the RIV set to the value indicated by “Frequency resource location of the initial transmission and retransmission” field in the configured sidelink grant. + For other contents of SCI format 1 + Set Modulation and coding scheme as indicated by high layers + “Priority” field is set according to the highest priority among those prioity(s) indicated by higher layers corresponding to the transport block + “Resource reservation” field shoud be set according to the followed Table based on indicated value $X$, where $X$ is equal to the Resource reservation interval provided by higher layer divided by 100

    UE procedure for receiving PSCCH

    UE procedure for determining resource block pool for PSCCH in sidelink transmission mode 3 and 4

PSCCH resource $m$ is the set of two contiguous resource blocks with the physical resource block number $n{PRB}=n{subCHRBstart}+m*n{subCHsize}+j for j=0 and 1$,where $n{subCHRBstart}$ and $n_{subCHsize}$ are given by high layer parameters startRBSubchannel and sizeSubchannel,respectively.

If PSCCH resource pool is configured, PSCCH resource $m$ is the set of two contiguous resource blocks with the physical resource block number $n{PRB}=n{PSCCHstart}+2*m+j for j=0 and 1$,where $n_{PSCCHstart}$ is configured by high layer parameters startRBPSCCHPool .
